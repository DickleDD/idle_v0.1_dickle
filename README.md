v0.3
** 本次改版重點放在職業定位、掉落體驗與卡利普袋子重製，讓玩家打寶、養成與推王路線更明確。

* 職業與技能調整

* 精靈魔法階級重新整理，移除無實質效果的空殼技能，並強化遠程 Buff 的實戰價值。風系弓箭流現在有更明確的命中與輸出提升。
* 黑暗精靈獲得吃王特化調整，新增破壞盔甲，並重寫雙重破壞與燃燒鬥志，使黑妖在挑戰 BOSS 時更穩定、更有爆發力。
* 騎士技能解鎖全面修正，高階技術書不再被職業限制卡住。騎士也能在後期學習部分基礎魔法，養成彈性更高。

* 裝備與掉落

* 大量原本未實裝或掉落率為零的武器、技能書與水晶，已依地圖與怪物定位重新分配。中低階地圖會有更多驚喜掉落，高階地圖與 BOSS 也加入更符合身價的稀有獎池。
* 部分無效技能書已從道具、商店、怪物掉落與袋子獎池中移除，減少玩家打到垃圾空殼道具的情況。
* 力量魔法頭盔調整為騎士專屬裝備，魔法書「體魄強健術」也加入小怪掉落。

* 卡利普袋子重製

* 卡利普袋子交換規則全面翻新，不再需要湊齊四種寶石。只要持有該品質任一寶石即可交換，系統會優先扣除數量最多的寶石。
* 三種袋子的獎池重新設計，依照普通、高級、稀有分層加入寶石循環、武器、技能書與箭桶。畢業級神兵維持 BOSS 專屬產出，不放入抽獎池。
* 袋子介面現在會顯示獎池機率，讓玩家交換前能更清楚知道可能開出什麼。

* 地圖事件

* 古魯丁地監 7 樓新增歐林彩蛋出現率調整。刷出歐林時，戰鬥資訊欄會出現專屬劇情對白。

* 新增一鍵出售功能
