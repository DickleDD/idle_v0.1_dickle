v0.7  
此次更新的核心目標是降低中後期卡牆感，同時保留放置遊戲長線累積、掉寶期待與職業路線差異。

* 【版本重點】本次更新集中調整中後期成長節奏、卡立普寶石交換、騎士武器路線分工、道具整理出口，以及中後期追裝體驗。
* 【等級成長】平滑 45～55 級經驗需求曲線，降低 48、49 級附近突然暴增的升級壓力。
* 【等級成長】47～50 級現在會更穩定推進，不再出現前一階還能成長、下一階突然卡死的斷層感。
* 【等級成長】保留 56 級以後的長期放置成長曲線，後期仍需要時間累積，不會變成短期速成。
* 【卡立普交換】降低卡立普袋子直接開出中後期關鍵武器的機率，避免玩家過早取得核心武器後失去追裝目標。
* 【卡立普交換】高級與稀有袋子的主要獎勵改為武器碎片、封印武器、鍛鋼、武器核心與製作材料。
* 【卡立普交換】稀有袋子仍保留極低機率開出部分成品武器，但不再作為主要取得來源。
* 【卡立普交換】基礎袋子的武器成品權重也同步下修，改加入卡立普武器碎片、卡立普鍛鋼與卷軸類獎勵。
* 【道具出口】新增卷軸、藥水、寵物復活卷軸、寵物進化果實等消耗品的販賣出口。
* 【道具出口】新增任務材料販賣出口，水龍鱗、火龍鱗、古代卷軸、封印裝備、卡立普碎片等材料可轉換為金幣。
* 【道具出口】任務材料新增丟棄功能，玩家可以清理暫時沒有用途或不想保留的材料。
* 【道具出口】背包新增一鍵出售「卷軸藥水」與「任務材料」按鈕，降低中後期背包整理負擔。
* 【黑暗妖精】破壞盔甲現在明確標示為 40 級可用，避免技能階級與可用等級不清楚。
* 【騎士路線】重新定位騎士單手劍與雙手武器路線，雙手武器維持高輸出與反擊核心，單手劍加盾牌改走防禦、穩定、控場與掛機安全路線。
* 【騎士雙手】反擊屏障維持雙手武器限定，仍是雙手騎士的代表性核心技能。
* 【騎士單手】新增「盾擊」技能，限定單手近戰武器加盾牌時使用。
* 【騎士單手】盾擊會造成小量傷害，並短時間降低怪物命中與攻擊節奏。
* 【騎士單手】盾擊對 Boss 改為弱化效果，仍可干擾攻勢，但控制與壓制強度低於一般怪物。
* 【騎士單手】新增「劍盾精通」被動技能，限定單手武器加盾牌時發揮效果。
* 【騎士單手】劍盾精通提供命中、傷害減免、魔法防禦與額外盾牌 AC 加成，強化單手盾騎士的穩定性。
* 【騎士防禦】堅固防護在單手武器加盾牌狀態下追加 DR 與 MR，提高面對高傷害怪物與魔法壓力時的掛機安全性。
* 【騎士防禦】增幅防禦在裝備盾牌時追加 AC 與 DR，使盾牌流派有更明確的防禦收益。
* 【騎士平衡】本次沒有大幅提高單手劍傷害，避免單手盾路線壓過雙手武器輸出定位。
* 【技能掉落】新增的盾擊與劍盾精通技能書已加入中階怪物掉落。
* 【技能掉落】奇岩周邊、海賊島、歐瑞周邊、火龍窟等中階到中後期區域可取得新的騎士單手盾技能書。
* 【掉落節奏】大幅提高中後期低機率武器的掉落底線，減少長時間沒有目標掉落的停滯感。
* 【掉落節奏】45 級以上怪物的多數非終局武器掉落率獲得補強，讓中後期追裝更有感。
* 【掉落節奏】終局傳說武器仍維持稀有定位，不會因本次調整變成大量掉落。
* 【追裝體驗】中後期玩家現在更容易取得可替換、可強化、可期待的武器進度，而不是只靠極低機率賭單一畢業裝。
