v0.4.1 

修復與調整：
1. 修正瑪那魔杖無法販售的問題。
2. 修正歐西斯弓因售價計算為負數而無法販售的問題。
3. 修正瑪那斗篷效果未實際生效的問題，現在會正確提供 MP+30 與魔力恢復+5。
4. 裝備詳細資訊現在會顯示更多實際生效的能力，例如力量、敏捷、智力、最大 MP、HP/MP 恢復等。
5. 大幅提升 BOSS 擊殺獎勵：
   - BOSS 金幣改為穩定掉落。
   - 依 BOSS 等級提高金幣倍率。
   - BOSS 擊殺新增保底掉落數量。
   - BOSS 會進行額外掉落抽選。
6. BOSS 掉落表會排除部分低價值白裝，降低打王掉落垃圾裝備的情況。
7. BOSS 掉落裝備變成祝福裝備的機率提高。

v0.4  
本次更新重點調整地圖難度曲線、BOSS 分布與裝備成長節奏，讓玩家掛機狩獵時更平順、不容易被跨等級 BOSS 突然秒殺。

-地圖與怪物

* 新增法利昂、巴拉卡斯、林德拜爾棲息地，四龍改為獨立挑戰區。
* 新增水晶宮殿，冰之女王移入此地並強化魔法演出。
* 調整海音湖岸、伊娃王國地監、火龍窟、象牙塔、拉斯塔巴德等地圖難度。
* 新增歐瑞周邊、歐瑞系列怪物與多種新怪物。
* 多名 BOSS 與怪物更名，並追加特殊魔法、壞刀、夥伴攻擊等戰鬥紀錄演出。
* 遺忘之島維持最高難度定位，部分古代 BOSS 更名。

  掉落與經濟

* 大幅補充各地圖技能書、魔法書、水晶、武器與寶石掉落。
* 調整裝備與技能書販賣金幣曲線，讓成長資源取得更合理。
* 精靈墓穴加入精靈水晶共通掉落池。
* 拉斯塔巴德加入黑暗精靈武器與水晶掉落池。

-裝備與套裝

* 調整部分裝備職業限制。
* 多羅皮帶、毆擊皮帶能力重新設定。
* 新增冰之女王套裝效果。
* 新增影子套裝效果。

-介面與體驗

* 地圖列表重新排序，水晶宮殿與四龍棲息地集中顯示於列表底部。
* 四龍與水晶宮殿加入特殊文字顏色標示。
* BOSS 魔法傷害加入最低傷害門檻，避免高 MR 角色將 BOSS 魔法壓到過低。
* 實裝二段加速。

v0.3  
本次改版重點放在職業定位、掉落體驗與卡利普袋子重製，讓玩家打寶、養成與推王路線更明確。

-職業與技能調整

* 精靈魔法階級重新整理，移除無實質效果的空殼技能，並強化遠程 Buff 的實戰價值。風系弓箭流現在有更明確的命中與輸出提升。
* 黑暗精靈獲得吃王特化調整，新增破壞盔甲，並重寫雙重破壞與燃燒鬥志，使黑妖在挑戰 BOSS 時更穩定、更有爆發力。
* 騎士技能解鎖全面修正，高階技術書不再被職業限制卡住。騎士也能在後期學習部分基礎魔法，養成彈性更高。

-裝備與掉落

* 大量原本未實裝或掉落率為零的武器、技能書與水晶，已依地圖與怪物定位重新分配。中低階地圖會有更多驚喜掉落，高階地圖與 BOSS 也加入更符合身價的稀有獎池。
* 部分無效技能書已從道具、商店、怪物掉落與袋子獎池中移除，減少玩家打到垃圾空殼道具的情況。
* 力量魔法頭盔調整為騎士專屬裝備，魔法書「體魄強健術」也加入小怪掉落。

-卡利普袋子重製

* 卡利普袋子交換規則全面翻新，不再需要湊齊四種寶石。只要持有該品質任一寶石即可交換，系統會優先扣除數量最多的寶石。
* 三種袋子的獎池重新設計，依照普通、高級、稀有分層加入寶石循環、武器、技能書與箭桶。畢業級神兵維持 BOSS 專屬產出，不放入抽獎池。
* 袋子介面現在會顯示獎池機率，讓玩家交換前能更清楚知道可能開出什麼。

-地圖事件

* 古魯丁地監 7 樓彩蛋出現率調整。

* 新增一鍵出售功能
